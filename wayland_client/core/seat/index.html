Module wayland_client::core::seat [−] [src]

Structures related to seats and inputs.

A Seat represents a set of input devices, with possibly any combination of a Keyboard, a Pointer and a Touch (not yet implemented). Most classic settings will have a single Seat.

These objects allow you to handle user input, using various callbacks.
